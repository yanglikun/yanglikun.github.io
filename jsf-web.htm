差旅WEB端JSF接口

引入

POM
  1.         <dependency>
  2.             <groupId>com.jd.travel</groupId>
  3.             <artifactId>jd-travel-rpc</artifactId>
  4.             <version>1.0-SNAPSHOT</version>
  5.         </dependency>

配置
测试
  1.         <jsf:consumer id="orderService"
  2.                       interface="com.jd.travel.rpc.service.OrderService"
  3.                       protocol="jsf"
  4.                       alias="testOrderService"
  5.                       timeout="10000">
  6.         </jsf:consumer>
测试环境如果想精确调用到测试环境可以用jsf直连的方式如下：
  1.     <jsf:consumer id="orderService" alias="testOrderService"
  2.                   interface="com.jd.travel.rpc.service.OrderService"
  3.                   url="jsf://192.168.147.89:22002" timeout="10000">
  4.     </jsf:consumer>
其中url中的端口，通过http://test.jsf.jd.com/可以查询出来
